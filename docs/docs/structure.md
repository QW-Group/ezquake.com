# Files and directory structure

## Quake

By default Quake is installed into `x:/quake`.

```
/quake
    quake.exe    
    
    /id1
        pak0.pak
        pak1.pak
```

`pak0.pak` and `pak1.pak` hold all the games' data like graphics, sounds, models and maps.

## ezQuake

```
/quake
    /id1
    /ezquake
        /configs
        /sb
    /qw
```
